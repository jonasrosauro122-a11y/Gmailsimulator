# Executive Inbox Simulator: Email Triage & Response Training

A GitHub-ready training project for Executive Assistant, General VA, and Admin Support email practice.

## What's Included
- `index.html` — clickable Gmail-style simulator
- `/scenarios/` — easy, normal, hard, and final assessment inbox scenarios
- `/templates/` — trainee response templates
- `/rubrics/` — trainer scoring guides
- `/model-answers/trainer-only/` — ideal answers for trainer reference
- `/trainer-dashboard/` — score tracker and coaching notes

## GitHub Setup
1. Create a new GitHub repository named `ea-email-simulator`.
2. Upload all files and folders from this package.
3. Open `index.html` locally or deploy to Netlify.
4. Trainees use the web simulator and scenario files to practice.
5. Trainers grade using the rubrics and score tracker.

## Netlify Setup
Build command: leave blank
Publish directory: `.`

This is a Gmail-style educational simulator. It is not connected to real Gmail and does not send real emails.
