# How to Use

1. Open `index.html`.
2. Use the clickable inbox simulator.
3. Choose a scenario in the Trainer Panel.
4. Draft an email in the compose window.
5. Click **Submit for Grade**.
6. Review the score and coaching notes.
7. Repeat until the trainee reaches 90% or higher.

## GitHub Practice
1. Trainer assigns a scenario from `/scenarios/`.
2. Trainee copies `/templates/response-template.md`.
3. Trainee completes the response.
4. Trainee saves it under `/user-responses/trainee-name/`.
5. Trainer grades using `/rubrics/`.
